a todolist written in **NodeJS** with rich features by [__*alone*__](https://www.iamalone.ir)




## Features ✨

- [X] User registration and login with JWT authentication  
- [X] Passwords hashed securely with Argon2  
- [X] Multi-user support  
- [X] Add, update, delete, and fetch tasks  
- [X] Task filtering by status, due date, and title  
- [X] Fully typed with TypeScript for safety  
- [X] Easy to run locally  



## Tech Stack ✅

- **Node.js** (v18+)  
- **Express.js**  
- **TypeScript**  
- **MongoDB + Mongoose**  
- **argon2** for password hashing  
- **JWT** for authentication  
- **Zod** for input validation  




## Installation 💿 
Clone Repo then run 

```bash
npm install
```

then create a env file and copy this in it

```js
PORT=3000 (You Can Change Port)

MONGO_URI=(Your MongoDb DateBase url)

JSON_SECRET=(Your Secret)

```

then  run 
```bash 
npm run dev
```
